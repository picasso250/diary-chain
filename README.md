# Diary Chain

[![Netlify Status](https://api.netlify.com/api/v1/badges/474a177d-1f29-4159-8de4-a071d637b237/deploy-status)](https://app.netlify.com/projects/diary-chain/deploys)

GitHub: [picasso250/diary-chain](https://github.com/picasso250/diary-chain)

Diary Chain is an on-chain diary application. It leverages the security and immutability of the Ethereum ecosystem.

## Tech Stack

### Frontend
- **Framework:** Svelte 5
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Web3 Integration:** ethers.js
- **Deployment:** Netlify (configured via root `netlify.toml`)

### Backend
- **Language:** Solidity

## Getting Started

### Prerequisites
- Node.js
- npm or yarn

### Frontend Setup

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
